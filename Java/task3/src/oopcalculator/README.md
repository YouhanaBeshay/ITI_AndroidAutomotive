# Task 3 : Simple OOP Calculator

A basic calculator that demonstrates OOP design principles using the Strategy and Factory design patterns.

## Features
- Addition, Subtraction, Multiplication, Division
- Clean separation of concerns via OOP
- Strategy pattern for interchangeable operations
- Factory pattern for operation creation and management

## Usage
- Compile : in `src` directory
    ``` bash
    javac oopcalculator/strategies/*.java oopcalculator/factory/*.java oopcalculator/*.java
    ```
- Run : in `src` directory
    ``` bash
    java oopcalculator.MyNewCalculator
    ```
