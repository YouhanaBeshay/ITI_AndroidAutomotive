# Calculator

A simple Android calculator built with Java.

## What It Does

- Basic math: add, subtract, multiply, divide
- Supports parentheses and decimal numbers
- Implicit multiplication: `2(3+4)` works as `2*(3+4)`

## Tech Stack

- **Language:** Java
- **Min SDK:** Android 10 (API 29)
- **UI:** LinearLayout, GridLayout, Material Components
- **Theme:** Dark mode (Catppuccin Mocha palette)
