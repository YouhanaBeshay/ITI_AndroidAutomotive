# Salary Estimator

A simple Android app that estimates salaries based on years of experience, job title, and country.

## What It Does

- Takes 3 inputs: Years of Experience, Job Title, Country
- Calculates an estimated salary using a hardcoded lookup table
- Applies a +3% multiplier per year of experience (capped at 15 years)

## Tech Stack

- **Language:** Java
- **Min SDK:** Android 10 (API 29)
- **UI:** ConstraintLayout, Material Components, ScrollView
- **Theme:** Dark mode first (Catppuccin Mocha palette)
