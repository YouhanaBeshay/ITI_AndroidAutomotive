# Explicit Intent Test App

A simple Android app that demonstrates passing data between two activities using explicit intents.

## What It Does

- **MainActivity** sends a message and count to **MainActivity2**
- **MainActivity2** displays the data sent by intent
- Tapping **Return** finishes Activity 2 and pops back to the main screen

## Tech Stack

- **Language:** Java
- **Min SDK:** Android 10 (API 29)
- **UI:** ConstraintLayout, MaterialCardView, Material Components
- **Theme:** Dark mode first (Catppuccin Mocha palette)
